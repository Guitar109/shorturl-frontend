# ShortURL Project
ShortURL is a URL shortening service built with React and Vite.

## Technologies
- **Frontend**: React
- **Build Tool**: Vite
- **Styling**: CSS,Bootrap,AntD
- **Version Control**: Git

### Steps to Install
1.cd shorturl-frontend 
2.npm install

### Run Project
npm run dev
